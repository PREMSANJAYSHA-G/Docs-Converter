Docs Converter - Ready-to-Run

1. Install Python 3.7+ if not installed.
2. Open terminal and go to this folder.
3. Install dependencies:
   pip install -r requirements.txt
4. Run the app:
   python app.py
5. Open browser:
   http://127.0.0.1:5000/
6. Upload Word or PDF files, convert, and download.
